如何用 ItemHub DIY 自動澆灌

材料

硬體

  • Arduino Nano 33 IoT
    • 功能: 單晶片具有 Wifi 功能, 把我們需要的資料送到 Server
  • 土壤濕度感測器
    • 功能: 土壤濕度感測器
  • 其他材料:
    • 麵包板
      • 功能: 比較好把所有元件串接起來
    • 沈水抽水馬達
      • 功能: 澆灌

軟體

  • Arduino IDE

硬體配置

運作模式

Arduino Nano 33 IoT D0 ↔ 抽水馬達 VIN
Arduino Nano 33 IoT GND ↔ 抽水馬達 GND
Arduino Nano 33 IoT A0 ↔ 土壤濕度感測 A0
Arduino Nano 33 IoT GND ↔ 土壤濕度感測 GND
Arduino Nano 33 IoT 3V3 ↔ 土壤濕度感測 VCC

步驟

設定 & 下載程式

  • 新增裝置
  • 將 D2 PIN 腳設定為開關, A0 PIN 腳設定微感測器, 下載程式碼

修改程式

  • 下載的檔案解壓縮並打開 *.ino 檔案
  • 修改 Wifi 帳密 
    #define STASSID "{SSID}"
#define STAPSK "{WIFI_PASSWORD}"
    把 {SSID} 改為 Wifi 的連線名稱
    把 {WIFI_PASSWORD} 改為 Wifi 的連線密碼
  • 找到以下的程式碼區塊 
    int value = digitalRead(pins[i].pin);
    改寫成讀取土壤濕度感測器的類比訊號 
    int value = analogRead(pins[i].pin);
  • 接著選擇晶片類型為 Arduino SAMD Boards > Arduino NANO 33 IoT 把剛寫好的程式燒錄到晶片上，這樣就完成了

ItemHub 觸發

  • 新增觸發, 當濕度感測器 (A0) 低於某一個數值的時候, ItemHub 就會將對應的 D2 腳位改成高電位
